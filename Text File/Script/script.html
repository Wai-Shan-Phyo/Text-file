<script> element

ကျွန်တော်တို့ရဲ့ html code ထဲကို JavaScript code တွေထည်ချင်ရင် <script> element နဲ့ထည့်ပါတယ် ok အဲ့တော့ ကျွန်တော်တို့မှာ code ရေးဖို့အတွက်နေရာရသွားပြီ ok အဲ့နေရာကြီးက element တစ်ခု ဖြစ်တယ်ဆိုတာ ကျွန်တော်တို့သိပြီဒါဆို ကျွန်တော်တို့ html elements တွေဆိုတာ <script></script> အဲ့လိုအဖွင့်အပိတ်နဲ့အထဲမှ ပါတဲ့ code တွေအကုန်လုံးပေါင်းမှ element လိုခေါ်တာသိတယ်အဖွင်အပိတ်တွေကို tag လို့ခေါ်တာသိတယ် ပြီးတော့ tag တွေမှာ attributes တွေရှိတာသိတယ် ဒါဆို ဘာ attributes တွေရှိလဲ

async
ဒီ tag က စပြီး download လုပ်ရမယ် ဒါပေမဲ့ page ထဲက တခြား action တွေကို မနှောင့်ယှက်သင်ဘူးပေါ့ ဒါက asynchronousပါ

charset
src ထဲက code က ဘယ် character set ကို သုံးသင်လဲ ဖော်ပြတာပါ ဒါပေမဲ့ အသုံးနည်းတဲ့ attribute တစ်ခုပါ

crossorigin
သူက script loading လုပ်တဲ့အချိန်မှာCORS ကို supportပေးတဲ့ သဘောပါCORS ဆိုတာ domainတစ်ခုကနေfiles တွေရယူဖို့သုံစတဲ့ standard machanism တစ်ခုပါ

deffer
သူက script ကို execute လုပ်တဲ့ အခါမှာ page ကို မထိခိုက်စေအောင်သုံးတဲ့။ attribute ပါ သူက ဘယ်လိုအလုပ်လုပ်သလဲဆိုရင် script ကို document ကို parse လုပ်နေတဲ့ အချိန်မှာ download လုပ်ပြီး parse လုပ်ပြီးတဲ့ အချိန်မှာ executeလုပ်ပါတယ် သူက external scriptတွေအတွက်ပဲသုံးပါတယ်

integrity
သူက external script တွေအတွက်ပဲသုံးတဲ့optional attribute တစ်ခုပါ သူက SRI လို့ခေါ်တဲ့ subsource integrity ကို verify လုပ်တဲ့အခါမှာသုံးပါတယ်
src
သူက external script တွေရဲ့ location ကို လှမ်းခေါ်ပေးတဲ့ optional attribute တစ်ခုပါ

type
Content type သတ်မှတ်တဲ့ attribute ပါ

အခုကျွန်တော်တို့ မှာ script နှစ်မိုးရှိပါတယ် internal scriptနဲ့ external scriptဖြစ်ပါတယ်
ဒါလဲ သိကြမှာပါ
Html ထဲက script element ထဲမှာ  ရေးထားရင် internal script အပြင်js file မှာ ရေးထားပြီး src attribute နဲ့ location ထောက်ပြီးပြန်ခေါ်တာပါ

Internal script
<script>
function sayScript() {
console.log("<\/script>");
}
</script>

External script
<script src="example.js"></script>

Ok အဲ့ဒါဆိုရင် ကျွန်‌ေတာ်တို့ scriptတွေကို ဘယ်နေရာမှာ ထားမလဲ

နှစ်နေရာရှိပါတယ်
Html head ထဲမှာ နဲ့
Body ထဲမှာ ထားနိုင်ပါတယ်

Head ထဲမှာ ထားခြာင်းအားဖြင့် ကျွန်ဝောာ်တို့က CSS နဲ့ တွဲထားတဲ့ ပုံစံဖြစ်တယ် ပြီးရင် သူက html page ကို render မလုပ်ခင် script ကို loading လုပ်တဲ့ သဘောမျိုးဖြစ်ပါတယ် အားနည်းချက်က  page loading လုပ်တဲ့နေရာမှာ delay ဖြစ်နိုင်ပါတယ်

<!DOCTYPE html>
<html>
<head>
<title>Example HTML Page</title>
<script src="example1.js"></script>
<script src="example2.js"></script>
</head>
<body>
<!-- content here -->
</body>

Body ထဲမှာ ထားခြင်းဖြင့် ပိုမြန်တဲ့ page loading ကိုရရှိပါတယ်

<!DOCTYPE html>
<html>
<head>
<title>Example HTML Page</title>
</head>
<body>
<!-- content here -->
<script src="example1.js"></script>
<script src="example2.js"></script>
</body>
</html>
