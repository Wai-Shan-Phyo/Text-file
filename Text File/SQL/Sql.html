What is SQL?
SQLရဲ့ အရှည်က Structured Query Languageဖြစ်ပါတယ်

SQL က Relational Database တွေနဲ့ လုပ်ကိုင်ဖို့ သုံးတဲ့ standard language ပါ

Most important SQL commands
SELECT - database ကနေ data ထုတ်ယူခြင်း
UPDATE - database ထဲမှ data အား update လုပ်ခြင်း
DELETE - database ထဲမှ data ကို ဖျက်ခြင်း
INSERT INTO - database ထဲသို့ data အသစ်ထည့်ခြင်း
CREATE DATABASE - database အသစ်တည်ဆောက်ခြင်း
ALTER DATABASE - databaseကို ပြုပြင်ခြင်း

CREATE TABLE -table တည်ဆောက်ခြင်းtable တည်ဆောက်ခြင်း
ALTER TABLE - table ကို ပြုပြင်ခြင်း
DROP TABLE - table ကို ဖျက်ခြင်း
CREATE INDEX - index တည်ဆောက်ခြင်း
DROP INDEX - Index ကိုဖျက်ခြင်း

SELECT syntax

SELECT column1, column2........ FROM table_name;

ဒါက SELECT ရဲ့ syntax ပါ
အကယ်၍ column အကုန်လုံးကို database ထဲက နေထုတ်ယူချင်တယ်ဆိုရင်
SELECT * FROM table_name;
လို့ရေး‌ရင် column အကြန်ပါလာမှာပါ

SELECT DISTINCT statement
သူက column ထဲက မတူတဲ့ data တွေကို ထုတ်ပေးပါတယ်
SELECT DISTINCT column1, column2,...... FROM table_name;

SELECT COUNT(column1, column 2,…)
သူက column ထဲက data row အရေအတွက်ကို ထုတ်ပေးပါတယ်
SELECT COUNT(column1, column 2,…)  FROM table_name;

SELECT COUNT(DISTINCT column1,column2,...)
သူက column ထဲမှာ မတူတဲ့ data ဘယ်နှ ခုရှိလဲဆိုတာကို ထုတ်ပေးတာပါ
SELECT COUNT(DISTINCT column1,column2,...)  FROM table_name;

WHERE

သူက database ထဲက table ထဲက data တွေကို filter လုပ်ဖို့သုံးတာပါထဲက table ထဲက data တွေကို filter လုပ်ဖို့သုံးတာပါ
SELECT column1, column2........ FROM table_name WHERE condition;

Conditions တွေက programming ကif else မှာ စစ်တဲ့ conditions တွေပါပဲ
Column=value
Column>value
Column<value
Column>=value
Column<=value
Column<>value (သူက not equal ပါတချို့ version တွေမှာ !=ကိုသုံးပါတယ်)
Column BETWEEN value AND value
Column LIKE pattern
Column IN (value1,value2,....)

AND OR NOT
ကျွန်တော်တို့ if else မှာ conditions နှစ်ခုစစ်တာတို့ ဘာတို့ဆိုရင် &&တို့ ||တို့ သုံးခဲ့သလို ပဲ ကျွန်တော်တို့ WHERE မှာလဲ ဒီလိုသုံးလို့ရပါတယ်
SELECT column1, column2........ FROM table_name WHERE condition1 AND condition2 AND ......... ;

SELECT column1, column2........ FROM table_name WHERE condition1 OR condition2 OR ......... ;

SELECT column1, column2........ FROM table_name WHERE NOT condition;

null or not

IS NULL က null ဖြစ်တဲ့ အခါမှာ ထုတ်ပေးစေချင်ရင် သုံးပါတယ်

IS NOT NULL ကပြောင်းပြန်ပေါ့

SELECT column1, column2........ FROM table_name WHERE column IS NULL/IS NOT NULL;

Example
SELECT column1, column2........ FROM table_name WHERE column IS NULL;

ORDER BY

သူကတော့ data တွေကို စီပြီးပြန်ထုတ်ပေးတာဖြစ်ပါတယ်

SELECT column1, column2........ FROM table_name ORDER BY column1, column2........ ASC/DESC;

Example
SELECT column1, column2........ FROM table_name ORDER BY column  ASC;
ဒါဆိုရင် column ကို A to Z စီမှာဖြစ်ပြီး  DESC ဆိုရင် Z to A စီမှာပါ

SELECT column1, column2........ FROM table_name ORDER BY column;
ဆိုရင် ASC အနေနဲ့ စိသွားမှာပါ

SELECT column1, column2........ FROM table_name ORDER BY column, column0;

ဆိုရင်column ကိုအရင်စီမယ် ပြိးရင် column ထြမျာ တူတဲ့ dataတွေကို column 0 အတိုင်းစီပြိး column 1 န‌ဲ့ column 2 ကို ထုတ်ပေးမျာပါ

SELECT column1, column2........ FROM table_name ORDER BY column ASc, column0 DESC;

ဒါဆိုရင် အပေါ်ကအတိုင်းပဲ ဒါပေမဲ့ column 0ကို စီရင် Z to Aစီ မှာပါ

INSERT INTO
သူက data ထည့်တာပါ
INSERT INTO table_name (column1,column2, .....) VALUES (value1,value2, .....);

Columnအကုန်လုံးကို ထည်တာဆိုရင်
INSERT INTO table_name VALUES (value1,value2, .....);

Update
Syntax
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;

သူက တော့ data update လုပ်တာပါ where နဲ့ data ကို filter လုပ်လိုက်ပြီး set ထဲကအတိုင်း column 1ထဲကို value 1,column2 ထဲကို value 2စသဖြင့်ထည့်သွားတာပါ

Delete
ဒီ statement ကတော့ table ထဲက data ကို delete လုပ်တာပါ
Syntax
DELETE FROM table_name WHERE condition;

Table ထဲက data အကုန် delete လုပ်ချင် ရင်
DELETE FROM table_name;

Limit
Limit ကတော့ data ဘယ်နှ ခုကိုဘဲ return လုပ်ပေးရမလဲသတ်မှတ်ပေးတာပါ
Syntax
SELECT column_name(s) FROM table_name WHERE condition LIMIT number;

Min and Max

Min ဆိုရင် ရွေးချယ်ထားတဲ့ column ထဲကအသေးဆုံး value ကို ထုတ်ပေးတာပါ max ကအကြီးဆုံးပေါ့
Syntax
SELECT MIN(column_name) FROM table_name WHERE condition;
