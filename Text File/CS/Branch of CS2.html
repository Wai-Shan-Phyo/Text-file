Communication and security

Networking
Protocol တွေဘယ်လို အလုပ်လုပ်သလဲ hardware level မှာ  data တွေဘယ်လိုပို.သလဲ။ ဘယ်လို  encode  လုပ်သလဲဆိုတာကို သင်တယ်။ ဥပမာ  TCP  ဆိုရင်ဘယ်လို  data ကိုသယ်နိုင်သလဲ သူက UDP  ထက်ဘာကြောင့် ပိုပြီး stable ဖြစ်တာလဲ။  TCP Protocol သုံးရင် ဘာတွေလုပ်ပေးရလဲ အစရှိတာတွေသင်တယ်။

Computer Security
ဒါကတော့ Application တွေ  device  တွေ protocolအစရှိတဲ့ ဟာတွေကို security ဘယ်လိုရှိသလဲ Attack vector တွေဘယ်လိုရှိသလဲ။ဘယ်လို ကာကွယ်ရမလဲ ဘယ်လို attack ဖြစ်နိုင်သလဲ ဆိုတာသင်တယ်။ နောက် Digital Forensic ဆိုတာရှိသေးတယ် သူကျတော့   Crime ဖြစ်ပြီဆိုရင် ဘယ်လို လိုက်မလဲ trace လုပ်မလဲဆိုတာကို သင်တယ်။

Cryptography
One way hash တွေ  encoding တွေ encryption  တွေ digital certificate တွေ အစရှိတာကိုသင်တယ်။ နောက်  Steganography ဆိုတာကျတော့   image သို.မဟုတ်  voice , media တခုခုမှာ ကိုထဲ့ချင်တာကို မမြင်ရအောင် ထဲ့ထားတာ။ Cryptography က ရလဲ ဖတ်မရအောင် ၀ှက်တယ်။ Steganography  ကျတော့ မမြင်ရအောင် hide တယ်ဆိုတဲသဘောပဲ။ Steganography ဘက်ဆိုရင် Dicrete Wavelet Trasformလိုကောင်တွေ ရှတယ်။ JPEG လိုကောင်တွေမှာ အဲ့ကောင်တွေနဲ.ဖွက်ရတယ်။  JPEG  သည် ခုနက wavelet တွေနဲ.မှ အဆင်ပြေတယ်။

Computer Graphic
သူကျတော့ 2D,3D computer graphic တွေဘယ်လိုအလုပ်လုပ်သွားသလဲသင်တာ။ ဥပမာ  အစက်ကလေး တစက်သုံးပြီး line  တလိုင်းရအောင် ဘယ်လိုဆွဲမလဲ။ စက်ဝိုင်းတခုဖြစ်အောင် ဘယ်လိုဆွဲမလဲ။ 2D Object  တွေကို ဘယ်လို rotate   လုပ်မလဲ။ 3D Object တွေကိုဘယ်လို  represent လုပ်သလဲ။Object tracking တွေဘယ်လိုလုပ်သလဲ အစရှိတာတွေကိုသင်တာ။
