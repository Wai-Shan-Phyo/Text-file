Branches of Computer Science.(Part 1)

Program ရေးတတ်တယ် Programming တတ်တာလောက်နဲ. အဟုတ်မှတ်နေလို. CS  ဆိုတာဘာလဲ ဘာတွေပါလဲ သိရအောင် ငထုံလေးတွေ များမလာအောင် ရေးပေးလိုက်ပါတယ်။

Mathematical foundations
Coding Theory
Data  ကိုဘယ်လို represent လုပ်မလဲ အဲ့လိုလုပ်ရင် minimum  information ဖြစ်အောင်ချုံ.တာတွေ ဥပမာ Compression Algorithm တွေမှာဆို  data ကိုချုံ.လို.ရအောင်လုပ်တာသည် Coding theory နဲ.ဆိုင်တယ်။ နောက် network ပေါ်မှာ data    ကို tranfer လုပ်တာမျိုးမှာ efficient  ဖြစ်အောင် ဘယ်လို encoding သုံးမလဲ ဒါမျိုးတွေမှာသုံးတယ်။ နောက် information theory သုံးတဲ့ machine learning လိုကောင်မျိုးမှာသုံးတဲ့ ဥပမာ Decision tree ဆို information theory က Gain Ratio  လိုကောင်တွေသုံးတယ်။

Game Theory
Game theory ဆိုတာ ၂ယောက် ဒါမှမဟုတ် အဲ့ထက်ပိုတဲ့သူတွေကြား decision လုပ်ရမှာ ဘယ်လို decison  ချမယ်ဆိုရင် အကောင်းဆုံးဖြစ်မလဲဆိုတာမျိုးကိုတွက်တာ။ ဥပမာ Min-Max algorithm မှာဆိုရင် player  2 ယောက်ရှိမယ် တယောက်က ဘယ်ကစားကွက်ကိုရွေ.ရင် (ဥပမာ chess  လိုကောင်မျိုး)ကိုက ဘယ်လော်  gain  ရပြီး opponent  ကဘယ်လောက် loss ဖြစ်မယ်ဆိုတာကိုတွက်တာမျိုး။AI မှာ agent တွေ cooporate လုပ်မယ်ဆိုရင် သူတို.အကျိုးရှိအောင်ဘယ်လိုတွက်လဲ ဥပမာ သူက action တခုလုပ်ရင် သူဘယ်လောက်ရမလဲ ဘယ်လောက်ဆုံးရှုံးမလဲ တခြားသူက မလုပ်ရင်ရော ဘယ်လောက်ရမလဲ ဒါတွေ ယှဉ်တွက်ရတာ တွေကိုသုံးပြီး decision  တွေ manangement လုပ်တာတွေ ပါတာမျိုးပေါ့။ ကစားတဲ့ Game မဟုတ်ပဲ ပြိုင်ဆိုင်ရတဲ့  win-lose situation မျိုးကို ဆိုလိုတာ။

Graph Theory
ဒါကတော့ရှင်းပါတယ်  Data structure  က  Graph  တွေနဲ.ပတ်သတ်တဲ့ minimum  spanning tree ရှာတာတွေ shortest route ရှာတာတွေ လုပ်တာပေါ့။ နောက်ပိုင်းကျတော့ social network တွေပါလာတော့ Graph mining  လုပ်ပြီး friend  တွေsuggest လုပ်တာ ကြော်ငြာတွေပြတာ အစရှိတာတွေပါ သုံးလာတယ်။

Mathematical Logic Boolean Logic
ဒါကတော့ boolean logic သုံးပြီး circuit gate တွေမှာသုံးတာ။ နောက် Logic programming မှာသုံးတာ။ နောက် Fuzzy Logic   သုံးပြီး AI  Soft Computing ဘက်မှာသုံးတာမျိုးပါတယ်။Logical Queries တွေ programming language   တွေရဲ. semantic တွေ sound ဖြစ်ကြောင်း  Proof ထုတ်တာတွေမှာလဲသုံးတယ်။

Number Theory
ဒါကတော့ အဓိက အားဖြင့် Cryptography မှာသုံးတယ်။   Prime number တွေ အကြောင်း natural number တွေအကြောင်းကို ပြောတာ။Probabilitsitc Number theory ဆိုရင်  machine learning မှာသုံးတယ်။

Algorithm
ဒါကတော့  sorting, searching, အစရှိတာတွေဘယ်လိုအလုပ်လုပ်သွားသလဲ တခုချင်းဆီမှာ O(n) နဲ.ဆို ဘယ်လိုရှိသလဲ။ ဘယ်လိုအခြေအနေမှာ ဘယ်လို algorithmသုံးသင့်သလဲ ။ Algorithm တခုချင်းဆီက ဘယ်လို  requirment တွေရှိသလဲ ဆိုတာကိုသင်တာ။ နောက်ပိုင်းမှာ Parallel Distribute Algorithm တွေပါလာတယ်။ခု  Big Data ဆိုရင်  Map Reduce  လိုကောင်တွေသုံးပြီး  parallel algorithm တွေလုပ်ကြတယ်။

Data Structure
ဒါကတော့ linked list, tree, graph အစရှိတာတွေကိုသင်တာ။ ကိုရဲ. Problem ကိုဘယ်လို data structure မျိုးနဲ. represent  လုပ်ရင်ပိုကောင်းမလဲ။  Acess, read, write, insert, delete အဲ့လိုကောင်တွေမှာ အချိန်ဘယ်လောက်ကြာသလဲ ။ဒါမျိုးတွေသင်တာ။

AI
AI အောက်က ဒါတွေထပ်ပါတယ်။
Intelligent ,rational behaviour ဖြစ်နိုင် တုပနိုင်တဲ့  Algorithm တွေ software တွေ အကြောင်းသင်တာ။

Automated Reasoning
Prolog  language မှာလို logic တွေသုံးပြီး reasoning လုပ်ခိုင်းလို.ရတယ်။ အဲ့လိုကောင်မျိုးတွေကိုသင်တာ။ ဥပမာ ဦးဘ သည် ယောကျာင်္း။ ဦးမြသည် ဦးဘ သား။ ဦး တင်သည် ဦးမြသား ဒါတွေသည် fact တွေ။ Logic ထဲမှာ အဖိုးဆိုတာ အဖေရဲ. အဖေ ဆိုတာကိုထဲ့ရေးထားရင် ဦးတင်သည် ဦးဘရဲ. မြေးဆိုတာကို အလိုအလျောက် reason လုပ်လို.ရသွားမယ်။AI ခေတ်ဦးပိုင်းမှာသုံးတယ်။ NLP  မှာလဲသုံးကြတယ်။စိတ်ဝင်စားရင် Visual Prolog ဆိုတာကို သုံးကြည့်။နောက် Mathematical logic တွေကိုရေးပြီး theory prover တွေဆောက်လို.ရတယ်။

Computer Vision
သူကတော့ ပုံတွေကို classification လုပ်တာ    Object တွေကို မှတ်သားအောင် မှတ်မိနိုင်အောင်လုပ်တာ။ ဥပမာ  CCTV   ပုံထဲကလူသည် ကိုရှာနေတဲ့လူလား  detect လုပ်တာ။ မျက်နှာတွေကို ပုံတွေမှာ detect လုပ်တာ။ ပုံထဲက မြင်နေရတာတွေသည် ဘာ  Object လဲဆိုတာကို ဆုံးဖြတ်တာ ဒီကောင်တွသည် Computer Vision ထဲမှာပါတယ်။စိတ်ဝင်စားရင် OpenCV ဆိုတာရှိတယ်။

Soft Computing
Fuzzy logic, Neural Network  အစရှိတဲ့ algorithm တွေကိုသုံးပြီး  AI machine learning ဘက်မှာ လုပ်တာကိုပြောတာ။ ဘာလို. Soft Computing လို.ခေါ်သလဲဆိုရင် တခြား Algorithm တွေလို အဖြေသည် input ဒါပေးရင်   output ဒါထွက်ရမယ်ဆိုပြီး ကွက်တိထွက်တာမဟုတ်ဘူး။ Imprecise  data, imprecise output ဖြစ်နိုင်တယ်။ ဒါကြောင့်  Soft Computing လို.ခေါ်တယ်။

Machine Learning
ဒါကတော့ Computer  တွေကို data ပေပြီး classification , clustering အစရှိတဲ့ အလုပ်တွေကိုသင်ပေးတာ။ Statistic, Information theory, Math လာတဲ့  theory တွေကို အခြေခံပြီး ထုတ်ထားတဲ့ Algorithm တွေကိုလေ့လာတာ။ နောက်ပိုင်း big data နဲ.တွဲပြီးလုပ်ချင်ရင် Apache Mahout, Spark တို.လိုကောင်တွေ ရှိလာတယ်။ Deep Learning ကလဲ ခုနောက်ပိုင်းအတော် popular   ဖြစ်လာတယ်။ Deep learning သည် Machine Learning နဲ.ဘာကွာသလဲဆိုတော့  many hidden layer တွေကိုသုံးတယ် နောက်ပြီး automatic feature selection ကိုပါလုပ်ပေးနိုင်တယ်။ အဲ့တော့ Data များများရှိရင် အဆင်ပြေပြီ။

Evolutionary Computation
Darwin theory ကိုအခြေခံပြီး သုံးတဲ့ Genetic Algorithm တို.   Differential Evolution  , Evolutionary Strategies တို.အစရှိတဲ့ Optimization  algorithm  တွေကိုပြောတာ။ သူတို.ကို Machine learning အပိုင်းမှာလဲတွဲသုံးတယ်။Natured Inspired Computing ဆိုတာလဲရှိသေးတယ်။ ဥပမာ  ပုရွက်ဆိတ်တွေ ပျားတွေ ငှက်တွေ အစရှိတာတွေကို တုထုတ်ထားတဲ့  Ant Colony Algorihtm, Bee Algorithm, PSO  အစရှိတာတွေ nature inspired computing လို.ခေါ်တယ်။ အုပ်လိုက် နေတဲ့ သတ္တဝါတွေကို အခြေခံပြီးထုတ်ထားလို.Swarm Algorithm လို.လဲခေါ်တယ်။

Natural Language Processing
ဒါကတော့ ဘာသာစကားတခုကနေ တခုကိုပြောင်းတာ။ စာသားကိုအသံထွက်ဖတ်နိုင်အောင် လုပ်ပေးတာ။ လက်ရေးတွေကို ကွန်ပျူတာနားလည်အောင်လုပ်ပေးတာ။ အစရှိတာတွေကို သင်တဲ့ဘာသာ။

Robotic
သူကတော့ Robot တွေဆောက်တဲ့နည်းပညာ။  Mechanical ရော computer vision, image, voice recognition  အစရှိတာတွေ AI , machine learning, search algorithm, planning  algorithm အစရှိတာတွေ ပါတယ်။
