Data P5

Data Structure (Part - 5)
====================

Stack
-----------
Stack ကတော့ one-dimensional array ကိုအခြေခံပြီး designed လုပ်ထားတဲ့ data structure ပါ။ သူ့ရဲ့ အဓိက လုပ်ဆောင်ချက်ကတော့ နောက်ဆုံးမှ၀င်တဲ့ data ကို အရင်ဖတ်တာပဲ။ ဘာနဲ့တူလဲဆိုတော့ ကျွန်တော်တို့ ပွဲဈေးတန်းမှာဆို တိုင်လေးတွေကို ကွင်းနဲ့ ပစ်ပစ်ထည့်ပြီး ဆော့ရတဲ့ကစားနည်းလိုပါပဲ။

အဲ့ကစားနည်းအကြောင်း ရှင်းပြရင် stack ကိုပိုနားလည်သွားပါလိမ့်မယ်။ ကျွန်တော်တို့ကို အရောင်မတူတဲ့ ကွင်းတွေပေးထားမယ် (ဥပမာ - အနီ၊ အဝါ၊ အပြာ၊ အစိမ်း၊ အဖြူ ထားပါတော့ ငါးကွင်းပေါ့) အဲ့အရောင်အစဉ်အတိုင်း ကွင်းတွေကို တိုင်မှာပစ်စွပ်လိုက်ရော။ အဲ့တာက data ၀င်သွားပြီလို့ နားလည်လိုက်။ ပြီးရင် အဲ့ကွင်းတွေကို ဆိုင်ရှင်က တခုစီ ပြန်ယူတဲ့အခါကျ အရင်ပစ်လိုက်တဲ့ကွင်းကို အရင်ယူလို့မရဘူး။ ဘာလို့ဆို သူ့အပေါ်မှာ နောက်မှပစ်လိုက်တဲ့ ကွင်းတွေထပ်နေလို့။ အဲ့တော့ နောက်ဆုံးမှ ပစ်လိုက်တဲ့ အဖြူကွင်းကို အရင်ယူရတယ်။ ပြီးရင် အစိမ်း၊ အပြာ၊ အဝါ၊ အနီ အဲ့လိုပေါ့။ အဲ့တာကို stack လို့ခေါ်တယ်။

အဲ့တာကြောင့် stack ကို last-in first-out (LIFO) system လို့လဲခေါ်တယ်။ ပြီးတော့ ဒီ stack data structure မှာ data store လုပ်တာကို push down (PUSH) လုပ်တယ်လို့ခေါ်ပြီး data ပြန်ဆွဲထုတ်တာကို pop up (POP) လုပ်တယ်လို့ ခေါ်တယ်။ အဲ့ဒီ့ push down နဲ့ pop up လုပ်တာကို ထိန်းချုပ်ထားတဲ့ variable ကို stack pointer လို့ခေါ်တယ်။

Queue
------------
Queue လဲပဲ one-dimensional array ကို အခြေခံပြီး designed လုပ်ထားတဲ့ data structure ပဲ။ stack နဲ့ ဘာကွာလဲဆိုတော့ သူက အရင်၀င်တဲ့ data ကို အရင်ဆွဲထုတ်တယ်။ ဘာနဲ့တူသလဲဆိုတော့ ATM တန်းစီသလိုပဲ။

ATM စက်ရှေ့ကို အရင်ရောက်တဲ့သူ အရင်ထုတ်ပဲ။ ထုတ်ပြီးရင် သူထွက်။ နောက်ရောက်တဲ့သူ ထပ်ထုတ်။ အဲ့လို အစဉ်အတိုင်းသွားတာ။

အဲ့တာကြောင့် queue ကို first-in first-out (FIFO) system လို့လဲခေါ်သေးတယ်။ Queue ကိုထိန်းချုပ်ဖို့အတွက် pointer နှစ်ခုလိုအပ်တယ်။ တစ်ခုက head ကို indicate လုပ်ထားတဲ့ front pointer နဲ့ တစ်ခုက tail ကို indicate လုပ်ထားတဲ့ rear pointer။

သူ့ operation အရ queue မှာ data store စလုပ်ပြီဆိုရင် tail မှာရှိတဲ့ rear pointer ကို +1 ပေါင်းပြီး data store လုပ်တယ်။ အဲ့ တာကို enqueue လုပ်တယ်လို့ခေါ်တယ်။ data တွေကို ပြန်ထုတ်ပြီဆိုရင် head မှာရှိတဲ့ front pointer ကို +1 ပေါင်းပြီး data ပြန်ထုတ်သွားတယ်။ အဲ့တာကို dequeue လုပ်တယ်လို့ခေါ်တယ်။
