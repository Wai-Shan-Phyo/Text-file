Data Structure P2
Data Structure (Part - 2)
====================
Basic Data Structure တွေက Programming Language တိုင်းမှာ ပါ၀င်ပါတယ်။ ဘာလို့ဒီလိုပြောရလဲဆိုရင် Basic Data Structure မှာ Integer, Character , Array လိုမျိုး Basic ကျတဲ့ data type တွေပါ၀င်လို့ပါ။ Problem-oriented data structure ဆိုတာကတော့ Problem တွေကို ဖြေရှင်းပေးရတဲ့ data structure ပါ။ သို့ပေမဲ့ Programming Language တွေမှာ မပါ၀င်တဲ့ Problem-oriented data structure တွေလဲရှိပါတယ်။ အဲ့ခါကျ Basic Data Structure တွေကိုပဲ ပြန်သုံးရပါတယ်။

Basic Data Structure ရဲ့ subitem လေးတွေကို အရင်ရှင်းပြပါမယ်။ Basic Data Structure မှာ data type သုံးမျိုးပါဝင်ပါတယ်။

1) Basic data type
2) Structure type
3) Abstract data type

တို့ပဲ ဖြစ်ပါတယ်။ Basic data type ဆိုတာ Data တစ်ခုချင်းစီကို သက်မှတ်တဲ့ နေရာမှာ သုံးပြီး program ရေးသားရာမှာ အသုံးပြုပါတယ်။  Basic data type ကို simple type နဲ့ pointer type ဆိုပြီး ထပ်ခွဲထားပါတယ်။ Simple type ဆိုတာကတော့ programming language တွေမှာ အများဆုံးတွေ့ရတဲ့ data type တွေပါ။ ယေဘူယျအားဖြင့် Programming မှာ simple type တွေကို သုံးပြီဆိုရင် ကိုယ်သုံးတဲ့ language ရဲ့ syntax rule နဲ့အညီ data type တွေကို declare လုပ်ရပါတယ်။ ဥပမာ -

Int x = 10;
String text = "Hello World";

Simple type မှာ

1) Integer type
2) Real number type
3) Character type
4) Logical type
5) Enumeration type
6) Partial type

တို့ ပါ၀င်ပါတယ်။

Integer type
----------------------
Integer type ဆိုတာ Integer တွေအတွက် သုံးတဲ့ data type ကိုပြောတာပါ။ ကိန်းဂဏန်းတွေထဲမှာမှ ဒဿမ မပါတဲ့ ကိန်းပြည့်တွေပေါ့။

Real Number Type
--------------------------------
Real Number type ဆိုတာလဲရှင်းပါတယ်။ real Number တွေကို ကိုယ်စားပြုတဲ့ data type ပါ။ ကိန်းပြည့်ကော ဒဿမကိန်းကော ပါ၀င်ပါတယ်။

Character type
--------------------------
Alphabets တွေ၊ numerals တွေ၊ symbols တွေကို character အနေနဲ့ သက်မှတ်ပြီး character type အနေနဲ့ အသုံးပြုပါတယ်။ character code တွေကို computer ထဲမှာ binary number တွေအဖြစ် ဖော်ပြပါတယ်။

Logical type
---------------------
Logical type ဆိုလဲ ရှင်းပါတယ်။ Logical Operations တွေဖြစ်တဲ့ AND, OR, NOT တွေကို ကိုယ်စားပြုပါတယ်။

Enumeration type
------------------------------
Enumeration type ဆိုတာကတော့ Variable တစ်ခုရဲ့ ဖြစ်နိုင်ခြေရှိတဲ့ values တွေကို ရေတွက်တဲ့အခါ သုံးရတဲ့ data type ပါ။ ဒီ enumeration type မှာ integer တွေကို နာမည်တပ်ပြီး အသုံးပြုပါတယ်။ ဥပမာ - gender data အတွက် enum data type သုံးမယ်ဆိုရင် ၀င်လာမဲ့ possible value သည် male, female, other တွေဖြစ်ပါတယ်။ အဲ့ possible value တွေကို integer အနေနဲ့ နာမည်တပ်လိုက်ပြီး (1: male, 2: female, 3: other) process လုပ်တဲ့အခါ integer နဲ့ပဲ လုပ်သွားပါတယ်။

Partial type
---------------------
Partial type ဆိုတာကတော့ ရှိနေတဲ့ original-value ကို အစိတ်အပိုင်းလေးတွေအဖြစ် ခွဲထုတ်ပြီး သက်မှတ်ပေးတဲ့ data type ကို ခေါ်ပါတယ်။ upper limit တွေ lower limit တွေကို constraints တွေအနေနဲ့ သက်မှတ်ပြီး ခွဲထုတ်ပေးတဲ့ data type ကို partial range type လို့ခေါ်ပါတယ်။

Pointer Type
------------------- ----
Pointer Type ဆိုတဲ့ သဘောကတော့ ဥပမာ  C Programming Language မှာ Variable တွေ, function တွေတည်ဆောက်ရင် သူတို့ရဲ့ address (တည်နေရာ) ကို Computer ရဲ့ Main Memory Unit မှာ တခါထဲ ခွဲဝေသက်မှတ်ပြီး သိမ်းထားပါတယ်။ အကျယ်ကို Pointer in C, C++ ဆိုပြီး ရှာဖွေ ဖတ်ရှုနိုင်ပါတယ်။
