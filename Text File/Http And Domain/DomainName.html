Domain Name and How it Works!

ဒီနေ့ လူတော်တော်များများမေးကြတဲ့ Domain အကြောင်းလေး ပြောပြပါမယ်။ ယနေ့ ခေတ်မှာ ဝန်ဆောင်မှုတွေ ပစ္စည်း တွေအကြောင်း သိချင်တဲ့အခါ များသောအားဖြင့် customer တွေက အရင်ဆုံး online ပေါ်က ရှာဖွေမှု ပြုလုပ်လေ့ရှိပါတယ်။ လုပ်ငန်းတစ်ခုက website ပေါ်မှာလွယ်လွယ်ရှာဖွေနိုင်ရင် ရောင်းချနိုင်တဲ့အခွင့်အလမ်းကို ပိုပြီးမြှင့်တင့်ပေးနိုင်ပါတယ်။ စီးပွားရေး လုပ်ငန်းတွေဟာ internet မှာ ဖြစ်ပေါ်လာတာနဲ့အမျှ ဈေးကွက်စီးပွားရေးစနစ်ဟာလည်း နောက်ပိုင်းကာလတွေမှာ သိသိသာသာ ပြောင်းလဲသွားခဲ့တာကိုတွေ့ပါတယ်။ ကိုယ့်ရဲ့ customer တွေဆီကို အထိရောက်ဆုံးနဲ့ မြန်မြန်ဆန်ဆန် ရောက်နိုင်ဖို့ website တွေကိုအသုံးပြုလာကြပါတယ်။

အဲ့လို website တွေကို သုံးလာတဲ့အခါမှာ ကိုယ့်ရဲ့business နဲ့ စပ်ဆက်တဲ့ မှန်ကန်တဲ့ domain name ကလဲ business ပုံရိပ်ကိုဖန်တီးတဲ့နေရာမှာ အရေးကြီးပါတယ် ။domain နဲ့ပက်သက်၍ ဆက်ပြီး ပြောပြပါရစေ။

Domain name ဆိုတာဘာလဲ ?
-----------------------------

Domain name ဆိုတာက website တစ်ခုကို ရှာဖို့  Browser URL မှာ ရိုက်ထည့်လိုက်တဲ့ website address ကိုပြောတာပါ။ ရှင်းရှင်းပြောရရင် website က အိမ်တစ်လုံးဆိုရင် domain name က အိမ်လိပ်စာပေါ့။

Computer တိုင်းမှာ IP address ရှိပေမယ့် IP address ကို အမြဲမှတ်မိနိုင်ဖို့က မလွယ်ပါဘူး။ IP address ဆိုတာ 66.249.66.1 စသဖြင့် ကိန်းဂဏန်းအများကြီးပါတဲ့ Address တစ်ခုဖြစ်ပြီး အကယ်၍ သင့်အနေဖြင့် သင်လိုချင်တာတစ်ခုခု ရှာဖို့ browser မှာ IP address ကိုသုံးပြီးသာ ရှာရမယ်ဆိုရင် မြင်ယောင်သာကြည့်ပါ။ ဖုန်းနံပါတ်တောင် များလာရင် မေ့လွန်းလို့ contact ထဲထည့်မှတ်ရတယ်မဟုတ်လား။ ဒါကြောင့် ဒီ IP address နဲ့ ရှာရင် ဖြစ်လာမဲ့ ပြဿနာတွေကို ရှောင်ဖို့ domain name ကို
တီထွင်ခဲ့ကြပြီး  တစ်ခုခု ရှာမယ်ဆိုရင် IP address ကို ရိုက်စရာမလိုတော့ဘဲ domain name ကို
ရိုက်ပြီး ရှာလို့ရသွားပါပြီ။ ဥပမာ Google website ကိုသွားချင်ရင် google.com ကို ရိုက်လိုက်ရုံပါပဲဗျ။

Domain name တွေ ဘယ်လို အလုပ်လုပ်သလဲ?
-------------------------------------------

သူကတော့ လက်တွေ့အပြင်မှာဆိုရင် ဖုန်းစာအုပ်လိုပါပဲ။ ဒါပေမယ့် Domain အကြောင်း ပြောမယ်ဆိုရင် အရင်ဆုံးDomain name system (DNS) အကြောင်း အနည်းငယ် သိဖို့လိုအပ်ပါသေးတယ်။ လက်ရှိ internet အသုံးပြုတိုင်းက DNS ကိုသိကြတဲ့ သူရှိသလို မသိတဲ့သူကလဲ ဒုနဲ့ဒေးပါပဲ။ DNS ဆိုတာ  သူတို့က domain name တွေအများကြီးကို manage လုပ်ရပြီး ၊ ပြီးလျှင်တော့ အဲ့ဒီ domain name တွေကို IP address အဖြစ်သို့ translate လုပ်ပေးလိုက်ခြင်းပါ ။

ဒါဆို Domain name ကကော ဘယ်လိုအလုပ်
လုပ်သလဲ ?
-------------------------------------------

Browser မှာ domain name တစ်ခု ရိုက်ပြီးတဲ့အခါcomputer ကနေ Server တွေကို Request
လှမ်းလုပ်လိုက်ပါတယ်။ ယင်း Request လုပ်လာတဲ့ domain name ကို DNS က IP address အဖြစ် translate လုပ်မယ် ထိုမှတစ်ဆင့် request လုပ်ထားသော web page နဲ့သက်ဆိုင်တဲ့ အချက်အလက်များကို browser သို့ ပြန်လည်ပေးပို့ခြင်းဖြစ်ပါတယ်။
